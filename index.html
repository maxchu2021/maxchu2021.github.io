關於我

您好，我是 Max Chu，從事網站開發經驗二十年，專注於前端技術 React / Vue 與後端技術 PHP （多種 MVC Framework），使用過 WordPress 和 Drupal 兩個熱門的 CMS 內容管理系統開發客製化網站。

曾任職於多間中小企業，有自行接案、和團隊開發產品、系統和平台、和參與過 AppWorks 的新創團隊一起共同創業，並設計開發 IoT 產品銷售到 Amazon 和北美市場，以及和美國團隊一起合作開發旅遊線上預訂包車平台...等經驗。

處理過無數多網站程式技術相關的疑難雜症，個人喜歡挑戰，因此也在每次進到新的領域時，都會積極學習和整合。

目前為自由工作者，提供網站開發服務和諮詢，歡迎合作邀約。

作品案例

VacayCar

VacayCar 是一個線上預訂行程的旅遊客製化網站，團隊中有美國人，需要使用到英語溝通，系統使用到 Yii PHP Framework 框架技術開發，提供行程導覽、自訂行程、線上預訂系統、旅遊部落格，方便使用者隨時自行規劃和預訂旅遊行程。

台灣步道守護網

使用到 PWA 技術開發的回報系統，在離線的狀態下能夠瀏覽使用網站，並用到 D3.js 的繪圖技術來呈現圖表，後台使用 Drupal 8 內容管理系統。

台灣昆蟲學會

使用 Drupal 8 內容管理框架獨立開發的台灣昆蟲學會官網，提供線上捐款、報名年會、發表論文讓系統匯出 Word 檔案和後台內容管理系統。
