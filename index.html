關於我

資深網站工程師，網站相關開發經驗二十年，專注於前端技術 React / Vue 與後端技術應用（多種PHP MVC Framework），使用過 WordPress 和 Drupal 兩個熱門的內容管理系統開發，習慣透過流程或新工具改善開發體驗。

曾任職於多間中小企業，有自行接案、和團隊開發產品和平台、以及和參與過 AppWorks 的新創團隊一起共同創業...等經驗。

處理過無數多程式技術相關的疑難雜症，個人喜歡挑戰，因此也在每次進到新的領域時，都會積極學習和整合。

目前為自由工作者，有和多家公司合作，提供網站開發的服務，歡迎合作邀約。
